Snipaste 是一款简洁高效的截图与贴图工具，支持丰富的标注功能和屏幕取色，支持 Windows 和 macOS。

## 核心功能

<Grid columns={2}>
  <NewCard icon="pencil" title="截图与标注" iconColor="#979BFF" size="sm">
    支持矩形、窗口、全屏截图，截图后可添加矩形、箭头、文字、马赛克等标注。
  </NewCard>
  <NewCard icon="document" title="贴图" iconColor="#FF9B6A" size="sm">
    将截图或剪贴板内容钉在屏幕上悬浮显示，可调整透明度和置顶。
  </NewCard>
  <NewCard icon="magnifier" title="屏幕取色" iconColor="#6ADFB8" size="sm">
    截图时可吸取屏幕任意位置的颜色值，支持多种颜色格式复制。
  </NewCard>
  <NewCard icon="laptop" title="历史记录" iconColor="#FFD166" size="sm">
    自动保存截图历史，方便随时回溯之前的截图内容。
  </NewCard>
</Grid>
