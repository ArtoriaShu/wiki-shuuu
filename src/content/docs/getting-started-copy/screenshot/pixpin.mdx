## 基础功能

- **截图**：支持矩形、窗口、全屏等多种截图模式
- **贴图**：将截图钉在屏幕上，方便对照参考
- **标注**：截图后可添加箭头、文字、马赛克、高亮等标注
- **OCR 文字识别**：识别截图中的文字并复制
- **长截图**：滚动截取长页面内容
- **GIF 录制**：录制屏幕为 GIF 动图
- **录屏**：录制屏幕视频
