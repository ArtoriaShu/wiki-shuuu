PixPin 是一款截图工具，集截图、贴图、标注、OCR、录屏于一体，界面简洁，操作流畅，支持 Windows 和 macOS。

经过多次迭代后，目前可以看作Snipaste的升级版。

## 核心功能

<Grid columns={2}>
  <NewCard icon="pencil" title="截图与标注" iconColor="#979BFF" size="sm">
    支持矩形、窗口、全屏、自由形状截图，截图后可添加箭头、文字、马赛克、高亮、计数标注等。
  </NewCard>
  <NewCard icon="document" title="贴图" iconColor="#FF9B6A" size="sm">
    将截图钉在屏幕上悬浮显示，方便对照参考，支持透明度调节和置顶。
  </NewCard>
  <NewCard icon="magnifier" title="OCR 文字识别" iconColor="#6ADFB8" size="sm">
    一键识别截图或图片中的文字，支持中英文混合识别，结果可直接复制。
  </NewCard>
  <NewCard icon="laptop" title="录屏 / GIF" iconColor="#FFD166" size="sm">
    支持录制屏幕视频和 GIF 动图，可选择录制区域，适合制作教程或演示。
  </NewCard>
</Grid>
