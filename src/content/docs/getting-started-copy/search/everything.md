## 基础功能

- **极速搜索**：基于 NTFS 索引，秒级搜索全盘文件
- **实时更新**：文件变动后索引即时更新
- **正则表达式**：支持正则表达式搜索
- **通配符**：支持 `*` 和 `?` 通配符
- **过滤器**：按文件类型、大小、日期等条件过滤
- **HTTP 服务器**：可开启本地 HTTP 服务，通过浏览器访问搜索
- **命令行支持**：支持命令行调用
