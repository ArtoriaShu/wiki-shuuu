InputTips 是一款轻量级输入法状态提示工具，在光标附近实时显示当前中英文状态，告别误输入，仅支持 Windows。

## 核心功能

<Grid columns={2}>
  <NewCard icon="information" title="状态提示" iconColor="#979BFF" size="sm">
    在光标附近实时显示当前输入法状态（中文 / 英文），直观清晰。
  </NewCard>
  <NewCard icon="setting" title="自定义样式" iconColor="#FF9B6A" size="sm">
    可自定义提示框的颜色、大小、位置偏移，适配个人习惯。
  </NewCard>
  <NewCard icon="laptop" title="多输入法兼容" iconColor="#6ADFB8" size="sm">
    兼容微软拼音、搜狗、百度等主流输入法，开箱即用。
  </NewCard>
  <NewCard icon="rocket" title="轻量后台运行" iconColor="#FFD166" size="sm">
    后台静默运行，占用资源极少，支持开机自启。
  </NewCard>
</Grid>
