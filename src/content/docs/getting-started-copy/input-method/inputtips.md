## 基础功能

- **状态提示**：在光标附近实时显示当前输入法状态（中文/英文）
- **自定义样式**：可自定义提示框的颜色、大小、位置偏移
- **多输入法支持**：兼容微软拼音、搜狗、百度等主流输入法
- **轻量运行**：后台静默运行，占用资源极少
- **开机自启**：支持设置开机自动启动
