# 布局参数说明

## 正文内容顶部间距

文件：`src/components/override-components/ContentPanel.astro` 第 8 行

```css
padding: 0.8rem var(--sl-content-pad-x) 1.5rem;
```

第一个值 `0.8rem` 控制正文标题和内容距离顶部的距离，调大往下移。

---

## 侧边栏内容顶部间距

文件：`src/components/override-components/PageFrame.astro` 第 161 行

```css
padding-top: 1.9rem;
```

控制侧边栏导航内容距离顶部的距离，调大往下移。
